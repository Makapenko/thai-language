# План: Добавление фраз урока 2

## Контекст

Урок 2 вводит **три новых концепции** поверх урока 1:
1. **Местоимения-дополнения** — новые местоимения (ดิฉัน, ท่าน, กู, มึง, เธอ), которые используются как объекты предложений
2. **Вопросительные слова** — ใคร, อะไร, เมื่อไหร่, ที่ไหน, ทำไม, อย่างไร — для открытых вопросов (не да/нет)
3. **Вежливые частицы** — ค่ะ (жен.), คะ (жен. вопр.) — в дополнение к имеющейся ครับ (муж.)

Глаголы и существительные берутся **из урока 1** — кумулятивно.

## Шаг 1. Раскомментировать данные в `vocabulary/lessons/lesson2.ts`

**Текущее состояние:** все массивы закомментированы (`// { thai: ... }`).

**Что сделать:** раскомментировать `pronouns`, `questionWords`, `particles`.

**Зачем:** без реальных данных генератор не сможет создать фразы. Сейчас файл — пустой скелет.

**Почему отдельный файл:** каждый урок добавляет только **новые** слова, а не дублирует старые. Функция `getCumulativeVocab()` из `aggregate.ts` объединит урок 1 + урок 2.

## Шаг 2. Добавить новые `SentenceType` в `types.ts`

**Текущее состояние:** `SentenceType` включает 10 типов без объекта + 8 с объектом. Вопросительных с вопросительными словами — нет.

**Что сделать — добавить 6 новых типов:**

```ts
// Открытые вопросы с вопросительными словами
| 'question_who'       // คุณเห็นใคร? — Кого ты видишь?
| 'question_what'      // คุณทำอะไร? — Что ты делаешь?
| 'question_when'      // เขาไปเมื่อไหร่? — Когда он поедет?
| 'question_where'     // เธออยู่ที่ไหน? — Где ты находишься?

| 'question_how'       // คุณไปอย่างไร? — Как ты поедешь?
```

**Зачем:** тип нужен для фильтрации паттернов в `generatePhrases()` и для типизации. Без этих типов генератор не узнает, какие паттерны использовать.

## Шаг 3. Создать `phrasePatterns.lesson2.ts` — новые паттерны урока 2

**Текущее состояние:** `phrasePatterns.ts` содержит только паттерны урока 1 (утверждения, отрицания, вопросы да/нет, времена).

**Что сделать:** создать новый файл с 6 паттернами вопросительных предложений.

### Паттерны с вопросительными словами

| Паттерн | Структура | Пример |
|---|---|---|
| `question_who` | subject + verb + **ใคร** | คุณเห็นใคร? |
| `question_what` | subject + verb + **อะไร** | คุณทำอะไร? |
| `question_when` | subject + verb + **เมื่อไหร่** | เขาไปเมื่อไหร่? |
| `question_where` | subject + **อยู่** + **ที่ไหน** | เธออยู่ที่ไหน? |
| `question_why` | **ทำไม** + subject + tense + verb | ทำไมเขาไม่มา? |
| `question_how` | subject + verb + **อย่างไร** | คุณไปอย่างไร? |

**Зачем отдельный файл:**
- `phrasePatterns.ts` уже большой (~300 строк). Добавление ещё 6 паттернов раздует его.
- Урок 3, 4, 5 тоже получат свои паттерны — логично иметь `phrasePatterns.lesson3.ts` и т.д.
- `phraseGenerator.ts` остаётся чистым — только функции, без данных.

### Вежливые частицы

Не создаём отдельные паттерны. Вместо этого — расширяем группу `ending` в `lesson2WordGroups` (ค่ะ, คะ). Генератор фраз не меняется — вежливая частица просто ещё один вариант в `ending`.
