# План исправления ошибок вопросительных фраз

## Проблема 1. «Где Я находится?» — неправильное спряжение глагола

### Описание
Появляется фраза **«Где Я находится?»**, хотя должно быть **«Где Я нахожусь?»**

### Корень проблемы

В паттерне `question_where` жёстко зашита форма глагола «находится» (3-е лицо ед.ч.):

```typescript
russianTemplate: (s, _v, qw) => `${qw.russian} ${getSubjectLabel(s)} находится?`,
```

Глагол `_v` игнорируется (подчёркивание означает «не используется»). Вместо спряжения по местоимению используется фиксированная строка.

**Результат:**
- ✅ «Где он находится?» — верно (3-е лицо)
- ❌ «Где Я находится?» — неверно (должно быть «нахожусь» — 1-е лицо)
- ❌ «Где Вы находится?» — неверно (должно быть «находитесь» — 2-е лицо мн.ч.)
- ❌ «Где Мы находится?» — неверно (должно быть «находимся» — 1-е лицо мн.ч.)

**Анализ других вопросительных паттернов:**

Все остальные паттерны **правильно используют спряжение**:

| Паттерн | RussianTemplate | Спряжение | Статус |
|---------|----------------|-----------|--------|
| `question_who_subject` | `v.present[2]` | 3-е лицо ед.ч. (фиксировано) | ✅ Верно (Кто = 3-е лицо) |
| `question_who` | `v.present[s.conjIndex]` | По подлежащему | ✅ Верно |
| `question_what` | `v.present[s.conjIndex]` | По подлежащему | ✅ Верно |
| `question_when` | `v.future[s.conjIndex]` | Будущее время | ✅ Верно |
| **`question_where`** | **`находится`** | **Фиксированная строка** | **❌ НЕВЕРНО** |
| `question_why` | `v.present[s.conjIndex]` | По подлежащему | ✅ Верно |
| `question_how` | `v.future[s.conjIndex]` | Будущее время | ✅ Верно |

**Вывод:** только ОДИН паттерн имеет эту проблему.

## Проблема 2. `getSubjectLabel` не распознаёт местоимения с уточнениями

### Описание
Функция `getSubjectLabel` добавляет маркер пола только для местоимений с точным `russian === 'Я'`:

```typescript
if (subject.russian === 'Я') {
  // добавляет (муж.) или (жен.)
}
```

Но местоимения из урока 2 имеют уточнения:
- `russian: 'Я (жен., форм.)'` — **НЕ равно** `'Я'` → условие не срабатывает
- `russian: 'Я (грубое)'` — **НЕ равно** `'Я'` → условие не срабатывает

**Результат:** выводятся громоздкие фразы типа «Где Я (жен., форм.) находится?» вместо «Где Я (жен.) находится?»

### Какие местоимения затрагивает

#### Урок 1 (все корректные):
| Местоимение | russian | getSubjectLabel возвращает | Статус |
|-------------|---------|---------------------------|--------|
| ผม | `Я` | `Я (муж.)` | ✅ |
| ฉัน | `Я` | `Я (жен.)` | ✅ |
| คุณ | `Ты` | `Ты` | ✅ |
| เขา | `Он` | `Он` | ✅ |
| เขา | `Она` | `Она` | ✅ |
| เรา | `Мы` | `Мы` | ✅ |
| คุณ | `Вы` | `Вы` | ✅ |
| พวกเขา | `Они` | `Они` | ✅ |

#### Урок 2 (проблемные):
| Местоимение | russian | getSubjectLabel возвращает | Статус |
|-------------|---------|---------------------------|--------|
| ดิฉัน | `Я (жен., форм.)` | `Я (жен., форм.)` | ❌ |
| ท่าน | `Вы (вежл.)` | `Вы (вежл.)` | ⚠️ (пока терпимо) |
| เธอ | `Ты (неформ.)` | `Ты (неформ.)` | ⚠️ (пока терпимо) |
| กู | `Я (грубое)` | `Я (грубое)` | ❌ |
| มึง | `Ты (грубое)` | `Ты (грубое)` | ⚠️ (пока терпимо) |

**Вывод:** критичная проблема только с местоимениями, начинающимися с `Я`, потому что для них должна добавляться метка пола.

## Рекомендуемое решение

### Для проблемы 1: Использовать спряжение глагола из объекта Verb

**Суть:** паттерн должен использовать массив `verb.present[]` с правильным спряжением по `conjIndex` из подлежащего.

**Изменения:**
- Вместо фиксированной строки `«находится»` использовать `v.present[s.conjIndex]`
- Это даст правильное спряжение: нахожусь / находишься / находится / находимся / находитесь / находятся

**Проблема:** глагол «находиться» **ОТСУТСТВУЕТ** в словаре урока 1 (`src/data/vocabulary/lessons/lesson1.ts`).

**Решение:** создать специальный Verb-объект для этого паттерна.

**Файлы для изменения:**
- `src/data/phrasePatterns.lesson2.ts` — паттерн `question_where`

---

### Для проблемы 2: Изменить проверку на `startsWith` вместо точного совпадения

**Суть:** проверять не `subject.russian === 'Я'`, а `subject.russian.startsWith('Я')`

**Изменения:**
- Заменить `if (subject.russian === 'Я')` на `if (subject.russian.startsWith('Я'))`
- Это захватит и `'Я'`, и `'Я (жен., форм.)'`, и `'Я (грубое)'`

**Файлы для изменения:**
- `src/data/phrasePatterns.ts` — функция `getSubjectLabel` (строка ~74)
- `src/data/phrasePatterns.lesson2.ts` — функция `getSubjectLabel` (строка ~30)

## Детальный план реализации

### Шаг 1. Исправить функцию `getSubjectLabel` в обоих файлах

**Файлы:** `src/data/phrasePatterns.ts` и `src/data/phrasePatterns.lesson2.ts`

**Текущий код:**
```typescript
function getSubjectLabel(subject: Subject): string {
  if (subject.russian === 'Я') {
    switch (subject.gender) {
      case 'masc':
        return `${subject.russian} (муж.)`;
      case 'fem':
        return `${subject.russian} (жен.)`;
      default:
        return subject.russian;
    }
  }
  return subject.russian;
}
```

**Новый код:**
```typescript
function getSubjectLabel(subject: Subject): string {
  if (subject.russian.startsWith('Я')) {
    switch (subject.gender) {
      case 'masc':
        return `Я (муж.)`;
      case 'fem':
        return `Я (жен.)`;
      default:
        return 'Я';
    }
  }
  return subject.russian;
}
```

**Ожидаемый результат:**

| Местоимение | russian | getSubjectLabel сейчас | getSubjectLabel после |
|-------------|---------|------------------------|-----------------------|
| ผม | `Я` | `Я (муж.)` | `Я (муж.)` ✅ |
| ฉัน | `Я` | `Я (жен.)` | `Я (жен.)` ✅ |
| ดิฉัน | `Я (жен., форм.)` | `Я (жен., форм.)` ❌ | `Я (жен.)` ✅ |
| กู | `Я (грубое)` | `Я (грубое)` ❌ | `Я (муж.)` ✅ |
